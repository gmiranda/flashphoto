Downloading :
You have different ways to get the CImg library :
  1. Download the latest stable package.

    The CImg library is provided as a .zip package that is plateform-independent. It contains the sources of the library, as well as various examples illustrating the use of the library functions and classes.

  2. You may be also more adventurous and use CVS. This ensures you will get the latest code available, and will ease the updates as well (knowing the tool 'cvs' is needed). To do this, just type the command :

    cvs -d:pserver:anonymous@cvs.sourceforge.net:/cvsroot/cimg co CImg

    in your favorite terminal. Nevertheless, you have to know that some code in the CVS version may be experimental, so always test the stable archive before reporting bugs.

  3. Pre-compiled examples for Windows: You may be only interested by a quick tour of what CImg can do, before downloading it. You can thus download the example executables only, for Windows-based OS :

    <Download the example executables only >

    Note that this archive doesn't contain any sources ! This will just give you a quick idea of what it's possible to do with the CImg library. You won't be able to use the library without the sources, available in the complete archive.
